## Technical Notes

### Database Entities
- Brand
- Product
- VoiceOfCustomer (gold nuggets, reviews)
- USP
- MarketingAngle
- Hook
- Template
- GeneratedAd
- Campaign (mirror of Meta)
- AdSet (mirror of Meta)
- Ad (mirror of Meta)
- PerformanceMetric

### Key Relationships
- Brand → many Products
- Product → many USPs
- USP → many MarketingAngles
- MarketingAngle → many Hooks
- Hook → many GeneratedAds
- Template → many GeneratedAds
- GeneratedAd → one Ad (when deployed)

### API Rate Limits to Consider
- Meta Marketing API: ~200 calls/hour/user
- OpenAI: Varies by tier
- Anthropic: Varies by tier
- Templated.io: Based on plan
